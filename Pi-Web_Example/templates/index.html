Pi-Web - Quick Reference Guide

This site, which is being hosted on a PiWeb Server, is a Quick Reference Guide to how to create a Pi-Web Server.

Getting the Package

The Pi-Web package can be found on Github at here. Download or clone the repositry, and copy and paste the "server" package into your own project folder.

Importing the Package

To import the main Server class use: from server import Server. You can also import the HTML Template Class with from server.html_template import HTML_Template. You do not need the HTML Template Engine but it makes it a lot easier to build webpages with HTML.

Using the Server Class to Create a Web Server - Creating Web Pages

Before you start a Web Server you need to create the web pages. But first you must create the Server Class. server = Server(Host IP, Port Number). For more information on choosing Port Numbers and Host IP Addresses see . So to create a Web Server on your local machine on the default port (Port 80); you would use server = Server("127.0.0.1", 80).

Now the instance of Server object can be used to add web pages to the server. To do this use it's add_page(Url Path, Page's HTML). So to create a web page at the server's url /index you would use server.add_page("/index", "<h1>Index</h1>").

Using the Server Class to Create a Web Server - Starting the Server

To start the server you need to use the start_server() method and to stop the server using the stop_server() method. So in the program as described you would use: server.start_server(); followed by server.stop_server(). The server will loop infinetly unless disrupted by the user but in case the start_server() method stops looping due to a bug or crash then the stop_server() method will clean up.

Checking to Make Sure it Works

To make sure the server works navigate to 127.0.0.1 in your address bar in your browser. (If you are using a different port to port 80 then it should be 127.0.0.1:port number/index). This should then display:

Index

Or alternatively any other HTML you specified in the second argument of server.add_page().

Rendering the HTML From a File

It is possible to write 100's of lines of html into a argument the create a web page but it would make much much more sence to write the HTML in an .html file, and then load it into the server to be sent to connecting clients. To do this Pi-Web contains a HTML template engine, that can load and render HTML from files. However, at the moment it is very primitive.
